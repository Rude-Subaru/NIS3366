## 📱 功能特性

### 💬 Chat — 智能对话
- 微信级非对称气泡排版，左侧 AI（蓝色头像），右侧用户（绿色头像）
- 历史上下文记忆，支持最近 N 条对话回溯

### 📅 Schedule — NLP 日程引擎
- 自然语言输入 → LLM 意图解析 + 纯手写 Regex 实体提取
- 日期 / 时间 / 事件极速剥离，拒绝沉重三方库
- 高危写入前强制唤起 ActionDialog，物理级拦截防越权

### 📚 Study — 进阶学习引擎
- 动态规划结构化技能学习路径，拒绝空洞闲聊
- 精准匹配知识点，生成可执行的学习推荐

### 💚 Psych — 毫秒级心理雷达
- 高频情绪词库精准匹配（`累`、`焦虑` 等）
- 绕开常规聊天流，触发底层专属安抚白名单

## 🛡️ 安全设计

1. **本地安全探针**：内置敏感词语料库，高效遍历匹配，前置拦截违规注入
2. **权限最小化原则**：`module.json5` 严格遵循按需申请，网络与本地存储隔离管控
3. **物理级操作拦截**：针对高危写操作，强制唤起 ActionDialog 进行用户确认

## 🚀 快速开始

### 环境要求
- DevEco Studio 3.1+
- HarmonyOS SDK API 9 / 10
- Node.js 14+

### 导入与运行
```bash
# 1. 使用 DevEco Studio 打开本项目根目录
# File -> Open -> 选择当前文件夹

# 2. 连接真机或启动模拟器

# 3. 点击 Run 按钮即可运行
```

### Mock 模式（开发/无网环境）
```typescript
// LLMService.ets
private USE_MOCK: boolean = true  // 无网环境自动降级为本地特征词拦截
```

Mock 引擎支持纯本地特征词阻击：
- `日程` / `会议` / `安排` → `action: "schedule"`
- `学习` / `课程` / `Python` → `action: "study"`
- `累` / `焦虑` / `好烦` → `action: "psych"`
- 其他 → `action: "chat"`

## 🎯 代表性 Action 返回格式

```json
{
  "action": "schedule",
  "reply": "已为您安排日程，请确认写入",
  "data": {
    "time": "明天15:00",
    "event": "开会"
  }
}
```

| Action | 触发场景 | 处理方式 |
|--------|---------|---------|
| `chat` | 普通闲聊 | 直通 ArkUI 渲染层 |
| `schedule` | 日程安排 | Regex 实体提取 → ActionDialog 确认 → 落盘 |
| `study` | 学习求助 | 结构化技能路径推荐 |
| `psych` | 情绪表达 | 触发心理卫士专属安抚逻辑 |
