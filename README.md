# 🤖 AI秘书 - 鸿蒙ArkTS大作业

> 2学分课程大作业 | 4人4周开发 | 极简架构

## 📱 项目简介

一个基于大模型的AI秘书应用，采用**读执行体-大模型-写执行体**架构，支持：

- 💬 智能聊天
- 📅 日程管理  
- 📚 学习推荐
- 💚 心理咨询

## 🏗️ 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                      前端 UI (ArkTS)                      │

└─────────────────────────────────────────────────────────┘
                           │
           ┌───────────────┼───────────────┐
           ▼               ▼               ▼
```

## 💡 核心设计

### Prompt设计
```
你是一个AI秘书助手...

【当前系统信息】
- 时间：2024-01-15 14:30
- 日期：2024/1/15 星期一

【对话历史】
...

【用户最新输入】
帮我安排明天开会

请按JSON格式返回：
{"action": "schedule", "reply": "好的", "data": "明天开会"}
```

### Action类型
| Action | 说明 | 处理方式 |
|--------|------|---------|
| chat | 普通聊天 | 直接回复 |
| schedule | 日程安排 | 弹窗确认 |
| study | 学习推荐 | Mock推荐 |
| psych | 心理咨询 | 直接回复 |

## 🔧 Mock模式

项目默认启用Mock模式，无需真实API Key即可运行：

```typescript
// LLMService.ets
private USE_MOCK: boolean = true  // 开发阶段保持true
```

Mock会模拟大模型返回，支持关键词识别：
- "日程"/"会议" → schedule
- "学习"/"课程" → study  
- "焦虑"/"累" → psych
- 其他 → chat
