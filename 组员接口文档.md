### 2️⃣ 组员B：数据库管理器 (`DatabaseManager.ets`)

#### 你的任务
- 将Mock内存存储替换为真实的HarmonyOS SQLite数据库
- 创建两个表：`Messages`（聊天记录）、`Schedules`（日程）

#### 已有代码
```typescript
// 当前使用内存数组模拟
private mockMessages: MessageRecord[] = []
private mockSchedules: ScheduleRecord[] = []
```

#### 需要完成

**1. 引入SQLite模块**
```typescript
import relationalStore from '@ohos.data.relationalStore'
```

**2. 实现 initDB()**
```typescript
initDB(): boolean {
  // 1. 获取RdbStore
  // 2. 创建 Messages 表
  // 3. 创建 Schedules 表
}
```

**表结构参考：**
```sql
-- Messages表
CREATE TABLE IF NOT EXISTS Messages (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  content TEXT NOT NULL,
  isUser INTEGER NOT NULL,  -- 0=false, 1=true
  timestamp TEXT NOT NULL
)

-- Schedules表
CREATE TABLE IF NOT EXISTS Schedules (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  content TEXT NOT NULL,
  createdTime TEXT NOT NULL,
  remindTime TEXT,
  isDone INTEGER DEFAULT 0   -- 0=false, 1=true
)
```

**3. 实现 CRUD 操作**
- `saveMessage()` - 插入聊天记录
- `getRecentMessages(count)` - 查询最近N条
- `saveSchedule()` - 插入日程
- `getAllSchedules()` - 查询所有日程
- `markScheduleDone()` - 标记日程完成

#### HarmonyOS SQLite 参考代码
```typescript
import relationalStore from '@ohos.data.relationalStore'

// 获取数据库实例
const STORE_CONFIG = {
  name: 'AIAssistant.db',
  securityLevel: relationalStore.SecurityLevel.S1
}

async initDB() {
  this.db = await relationalStore.getRdbStore(context, STORE_CONFIG)
  
  // 创建表
  await this.db.executeSql(`
    CREATE TABLE IF NOT EXISTS Messages (
      id INTEGER PRIMARY KEY AUTOINCREMENT,
      content TEXT NOT NULL,
      isUser INTEGER NOT NULL,
      timestamp TEXT NOT NULL
    )
  `)
}

// 插入数据
async saveMessage(content: string, isUser: boolean) {
  const valueBucket = {
    content: content,
    isUser: isUser ? 1 : 0,
    timestamp: new Date().toISOString()
  }
  await this.db.insert('Messages', valueBucket)
}

// 查询数据
async getRecentMessages(count: number) {
  const predicates = new relationalStore.RdbPredicates('Messages')
  predicates.orderByDesc('id').limit(count)
  const resultSet = await this.db.query(predicates)
  // 解析resultSet返回数组
}
```

## 💡 开发Tips

### 1. Mock模式
每个模块都有Mock实现，你可以：
- 先测试整体流程是否正常
- 再逐步替换为真实实现

### 2. 调试技巧
```typescript
console.info('日志信息')
console.warn('警告信息')
console.error('错误信息')
```

### 3. 权限配置
需要在你负责的模块使用某些API时，在 `module.json5` 中添加权限：
```json
{
  "requestPermissions": [
    { "name": "ohos.permission.INTERNET" },
    { "name": "ohos.permission.GET_NETWORK_INFO" }
  ]
}
```
